Basic Tags

Tag Name Description
<html> </html> HTML Creates an HTML document
<head> </head> Head Stores title and meta data that isn't displayed
<body> </body> Body Stores the visibile portion of the site
<title> </title> Title Puts name of document in title bar, showing on tabs and bookmarks

Body Attributes

Tag Name Description
<body bgcolor=?> Background Color Sets background color, using name or hex value
<body text=?> Text Sets text color, using name or hex value
<body link=?> Link Sets color of links, using name or hex value
<body vlink=?> Visited Links Sets color of visited links, using name or hex value
<body alink=?> Active Links Sets color of active links, using name or hex value

Text Tags

Tag Name Description
<pre> </pre> Preformatted Text Creates preformatted text
<h1> </h1> - <h6> </h6> Headline Creates headlines -- H1= largest, H6 = smallest
<b> </b> Bold Creates bold text (should use <strong> instead)
<i> </i> Italicize Creates italicized text (should use <em> instead)
<tt> </tt> Typewriter-Style Creates typewriter-style text
<code> </code> Code Used to define source code, usually monospace
<cite> </cite> Citation Creates a citation, usually processed in italics
<address> </address> Address Creates address section, usually processed in italics
<em> </em> Emphasize Creates italicized text
<strong> </strong> Strong Creates bold text
<font size=?> </font> Font Size Sets size of font, 1 - 7 (should use CSS instead)
<font color=?> </font> Font Color Sets font color (should use CSS instead)
<font face=?> </font> Font Face Defines the font used (should use CSS instead)

Links

Tag Name Description
<a href="URL">clickable text</a> Hyperlink Creates a hyperlink to a Uniform Resource Locator
<a href="mailto:EMAIL_ADDRESS">clickable text</a> Mailing Hyperlink Creates a hyperlink to an email address
<a name="NAME"> Target Location Creates a target location within a document
<a href="#NAME">clickable text</a> Target Location Link Creates a link to a target location
